## Future

本レポートのニュースから導出した 3 点の予測。

1. **「1M context が既定」のオープン重み時代に 2026 Q3 以降完全移行** — DeepSeek V4 の「1M context をボルトオンではなくゼロから設計」方針、Qwen3.6-27B の 262K→1M、Claude Opus 4.7 の 1M token context（Bedrock）、GPT-5.5 の長文脈強化は同じ方向。2026 Q3 までに「native 1M context」がオープン重みの最低ラインとなり、32K-128K 止まりの既存モデルはレガシー扱いに転落。RAG パイプラインは「検索→コンテキスト挿入」から「全データをそのまま投入＋動的プルーニング」方式に主流シフトし、ベクトル DB ベンダは「長 context キャッシュ最適化層」として再定義を迫られる。
2. **「Physical AI の 8 時間本番稼働」が 2026 後半のエンタープライズ調達条件に** — Siemens–Humanoid の HMND 01 Alpha が Erlangen で「デモではなく本番ライン」で **8 時間 / 90% 成功率 / 60 tote/h** を達成したことは、「ヒューマノイド = PoC」の段階からの明確な卒業点。2026 Q4 までに大手製造業（VW / Bosch / Mercedes / Foxconn / TSMC / トヨタ）の RFP から「24 時間連続稼働実績」「1 週間 MTBF」「90%+ 自律完了率」「既存 MES / PLC 統合」が事実上の足切り条件に組み込まれ、Physical AI の競争軸は「見栄えの良いデモ動画」から「既存 OT 環境への差し込み容易性」に完全に移る。
3. **「AI エージェントに displace されるエンタープライズ SaaS」の金融市場評価が 2026 Q2 中にリセット** — ServiceNow の Apr 23 の **-18%（過去最悪の 1 日下落）** と IBM の同日下落、Salesforce / Workday / Oracle の連鎖下落は、「AI のコスト削減恩恵 < AI による既存ワークフロー自動化による displace」への資本市場の評価軸シフトの現れ。2026 Q2 中に Gartner / Forrester の Magic Quadrant / Wave 系レポートで「Agent-native vs Legacy SaaS」の二軸化が明示され、時価総額評価は「シート数 × ARPU」から「エージェントが触る API 呼び出しの占有度」に移行。アナリストコンセンサス更新の遅行により、向こう 2 四半期で同様の 15%+ 暴落が Workday / Atlassian / Box / Smartsheet など 3-5 社で再発。
