Three predictions derived from this report.

1. **"Hyperscaler-AI-lab capital coupling" graduates from finance pact to compute-stack lock-in by Q3 2026.** Apr 24's Google **$40B** to Anthropic + the prior Amazon **$25B** + Microsoft's existing $13B+ to OpenAI mean the **top 3 frontier labs each have a hyperscaler underwriter on the cap table** — and the underwriting is paid partly in **GPU/TPU compute commitments** (Google's $40B is "cash and compute"). By Q3 2026, **frontier-model serving topology becomes dictated by funding origin**: Anthropic Claude Opus 4.7 / 4.8 ships natively on Trainium2 + TPU v8 (Amazon + Google paths), OpenAI GPT-5.5 / 6 stays Azure-first, with Chinese frontier labs (DeepSeek V4 / Qwen 3.6-3.7) doubling down on Huawei Ascend (per the Apr 26 Bloomberg-CCTV piece). The "model-portability promise" of 2024-2025 quietly ends; **enterprise multi-cloud LLM strategy shifts to "multi-frontier-vendor" rather than "multi-runtime."**
2. **Inference servers become the new "primitive supply chain" — Q2 2026 sees the first AI Infra-specific CVE class (LLM-INFRA-CWE) with mandatory NVD tagging.** The Apr 19-26 sequence — **LMDeploy CVE-2026-33626 SSRF (12h31m exploited)** + **SGLang CVE-2026-5760 (CVSS 9.8) Jinja2 SSTI RCE** + Pillar's **Antigravity sandbox-escape** + the Apr 21 Comment-and-Control prompt-injection chain across Claude Code / Gemini CLI / Copilot — compose a clear "inference-runtime + agentic-CI" attack surface category. By Q3 2026, NIST/NVD adds **a dedicated "AI-Infra" tag** to its CVE schema, OWASP's LLM Top 10 v2026 promotes **"Inference Server Template / Loader RCE"** to a top-3 entry, and supply-chain SBOM tooling (Snyk / Wiz / Pillar / Lasso / Lakera) ships **per-engine signature feeds for vLLM / SGLang / LMDeploy / Triton / Ollama** as standard SKUs.
3. **The "consumer-GPU coding agent" displaces the cloud-only coding API for ≥30% of dev workloads by Q3 2026.** Apr 22's Qwen3.6-27B (dense, beats 397B MoE on SWE-bench, **40 tok/s on RTX 3090 at Q4_K_M ~16.8 GB**) + the same-week Apr 23 Ollama MLX preview (1.57x prefill / 1.93x decode on M5 Max for Qwen3.5-35B-A3B) + Apr 25's DeepSeek V4-Pro 75%-off API (effectively 28x cheaper than Claude Opus 4.7) compose a **3-corner squeeze on the proprietary frontier-API SaaS model**. By Q3 2026, **enterprise dev tools default to a hybrid harness** — local Qwen 3.6-class for "coding, refactor, lint, test" + remote frontier API only for "long-horizon planning / multi-tool reasoning" — and all Big-3 IDEs (Cursor / Windsurf / Kiro Power) ship a **local-first toggle** by default. "Cloud-only coding agent" becomes a 2024-era artifact.
