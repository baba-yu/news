## Relation to My Own Predictions

### 1. 「Malicious local LLMs will begin to function like malware」

- **Anthropic MCP Design Vulnerability（OX Security 開示、Apr 23-24 拡散）— 200,000 サーバ RCE リスク、設計レベル欠陥** — Anthropic MCP SDK の STDIO interface が「設定 → コマンド実行」を直接許可する設計欠陥、Python / TypeScript / Java / Rust 全実装が影響、150M ダウンロードに波及。OX Security が 11 MCP レジストリに poison test payload 投入で 9 件成功、6 件本番プラットフォームで command execution 確認。**Anthropic が「expected behavior」として CVE 否認** という対応はユーザ予測の「振る舞い監視・管理ソリューションは短期注目に留まり、抜本対策にはならない」傾向と整合（設計レベル欠陥はパッチでも振る舞い監視でも閉じない）。研究チームが個別パッケージで CVE-2026-30623 (LiteLLM) / CVE-2026-33224 (Bisheng) 等 10+ 件発行している点は「multilayer authentication for data access」型対応への移行を示唆。参照: [The Hacker News - Anthropic MCP Design Vulnerability](https://thehackernews.com/2026/04/anthropic-mcp-design-vulnerability.html)
