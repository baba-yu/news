## Relation to My Own Predictions

3つの個人予測のうち2つが本日強く裏付けられた。

**悪意ある local LLM がマルウェアのように機能する → 強い裏付け。** SGLang CVE-2026-5760 のケースは、まさに記述された失敗モード: モデル・ファイル (GGUF) が実行コード (`tokenizer.chat_template` 内の Jinja2 SSTI) を運び、推論サーバ内で発火する。`github.com/Stuub/SGLang-0.5.9-RCE` の公開 PoC + CERT/CC VU#915947 で、理論ではなく武器化された段階に到達。CERT/CC は調整プロセスで上流パッチが得られなかったと報告しており、これは予測の「基盤はゼロトラストでなければならない」側を補強 — 振る舞い監視による防御では SSTI レンダリングは捕まらなかったはず。Showcase 2026 の Okta blueprint はエージェントへゼロトラスト/多層認証を明示適用 (Agent Gateway = 中央 MCP コントロールプレーン、credential vaulting + 自動ローテーション、universal logout によるランタイム強制)、TrueFoundry の「MCP Security: Guide to Zero Trust for Agentic AI」がこれを形式化 — 「AI 単独ではアクセス不可能なシステム経路を意図的に設計する」アーキテクチャが現実の製品カテゴリになりつつある。ハーネス・エンジニアリング側では、Anthropic の 4/23 Claude Code 後付け検証 (3/4 推論カット + 4/16 verbosity-prompt 問題) と 4/24-25 v2.1.118-120 の24時間リグレッション・クラスタが behavior-spec-as-defense の脆さを示唆 — 正規のハーネス変更でさえ品質をサイレントに劣化させる、これは予測の「監視・管理は短期的注目のみ」フレーミングと整合的。

**クラウド API for 高度作業 + ローカル LLM for ルーチン作業の役割分担 → 直接的な明文化確認。** 本日の Future 予測 #3 は「Q3 2026 までに consumer-GPU コーディング・エージェントが cloud-only コーディング API を開発ワークロードの ≥30% で置き換える」と明示し、Qwen3.6-27B + Ollama MLX + DeepSeek V4-Pro 75% 値下げ API の3点同時圧迫がドライバ。価格構造シフト側も着地: GPT-5.5 API は入力 $5 / 出力 $30 で GPT-5.4 比2倍値上げ (Codex は出力トークンを ~40% 削減して「per-task コストで per-token 値上げを相殺」) — 先行シグナルとして指摘した SaaS 価格構造変化が、DeepSeek V4-Pro が Claude Opus 4.7 比 ~7x 安価で出荷された同じ週に発現。Tesla の Q1 2026 capex 上方修正 ($25B 超、carry-forward 株価セクション参照) は AI コンピュート・コスト曲線上の並行データ点。
