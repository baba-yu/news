## Summary of Findings

本日のレポートは、過去1週間の予測のうち、エージェント制御プレーンと AI-Infra-CVE-class 予測に対する単日最強の確認シグナルを示している。Microsoft Agent 365 の 5/1 GA(M365 E7 内 $15/user/月、テナントレベルでの MCP サーバー allow/block + Entra/Purview/Intune/Defender 統合)は、2026-04-25 予測が名指しした「MCP Gateway / MCP Firewall」エンタープライズプリミティブをそのまま具現化したものであり、その予測の Q3 ウィンドウより 4 ヶ月前倒しで着地している。5/1 の CSA「Identity in the Age of AI」公開と、CSAI Foundation MITRE-CNA + ATT&CK v19 命名技術波の継続を組み合わせると、MCP プロトコル攻撃面 / agent-skills 脅威マトリクス / AI-Infra-CVE-class クラスター全体が予定通り具現化している。

KV-cache-compression-as-first-class-knob 予測(2026-04-29 と 2026-04-30)が 2 番目に強いクラスター。本日は既存の TurboQuant + PolyKV + kvpress + Unsloth + SGLang #21419 / #23134 + llama.cpp #20969 スタックに CodeComp / CacheFlow / TokenDance / TTKV を追加。arxiv 波が代替案を広げており(本予測は明示的に「TurboQuant + ≥1 alternative」を要求)、Foundry Local が依然として欠けている Windows-default 統合点。

Anthropic Opus 4.7 GA が Claude products + API + Bedrock + Vertex AI + Microsoft Foundry に展開している(本日継続)のは、2026-05-01 マルチクラウドフロンティアモデル SLA 予測に最もきれいに合致する:OpenAI 以外の単一フロンティアラボが 4 つすべての主要ホスティングパスにまたがって出荷しており、これは予測が 4/27 Microsoft-license-non-exclusive 再構築によって開かれた white-space と呼ぶ「frontier on every cloud」パターンそのもの。

Pentagon-Anthropic 分裂 + 7 ベンダー機密ネットワーク契約は、2026-05-01 hyperscaler-AI-lab capital-coupling 予測の **procurement-precedent** 層での部分的確認(契約再構築層ではない);連邦判事が阻止した Trump フリーズ + 4/17 Susie Wiles / Amodei 会談は、本予測が Q3 決算までに各フロンティアラボ取締役会へ正式採用を強制すると見る deal-or-no-deal バイナリを設定する。

## Relation to My Own Predictions

本日のニュースには、3 つの定常予測のうち 2 つと強い重なりがある。

**悪意のあるローカル LLM / 根本的安全策としてのゼロトラスト** 予測は本日のニュースで直接追跡されている。Microsoft Agent 365 GA は「AI 単体ではデータにアクセスできないシステムパスを意図的に設計する」アーキテクチャを主流のエンタープライズ価格に置く——observe / govern / secure 三本柱 + Entra ID + Purview + Intune + Defender + テナント単位の MCP サーバー allow/block。同伴の CSA ブログ「Identity in the Age of AI: Rethinking Zero Trust's First Pillar」(5/1)は AI エージェントに対するゼロトラストの実効的な第一の柱がアイデンティティであると明示的に主張しており、これは本予測が指摘するフレーミングそのもの。本予測が短期的注目を集めるが回避リスクを覆さないとする振る舞い監視側も、本日 Agent 365 の「observe」サーフェス第二の柱(agent inventory + observability dashboards + policy templates)と CSAI Foundation の STAR-for-AI Catastrophic Risk Annex(2026年6月→2027年12月の4段階展開)を通じて可視化されている。本予測が短期・長期ともに安定すると述べる経路——AI へのゼロトラストベースのアクセス制御——は、Microsoft が $15/user/月で固定しているまさにその経路。本予測が短期的にのみ注目を集めるとする経路——振る舞い監視/管理——も出荷されてはいるが、基盤ではなく上位レイヤとして。本日のニュースは本予測の構造的賭けと整合する。

**高度な知的タスクはクラウド API / 日常運用はローカル LLM + クラウド SaaS 価格上昇** 予測は部分的に追跡されている。ローカル LLM スタックが本日成熟を続けている(DeepSeek V4 Pro/Flash 1Mコンテキスト + Hybrid Attention + Huawei Ascend 最適化、V4 Flash $0.14/$0.28 per M tokens;Qwen 3.6;Kimi K2.6 が SWE-Bench Pro で GPT-5.4(xhigh)を初めて上回るオープンウェイトモデルに;GLM-5.1 が SWE-Bench Pro 首位;Bonsai-8B 1-bit が iPhone 17 Pro Max で 44 tok/s;Foundry Local;Unsloth 2026.4.8)——いずれも「日常運用はローカル LLM」レッグ。クラウド SaaS 価格上昇レッグは本日直接証拠が弱い:データポイントは Microsoft Agent 365 が $15/user/月で価格設定(M365 上に重なる *新規* per-user 行であり、既存サービスの値上げではない)、AAPL Services $30.97B(+16%)(消費者側のサービス価格決定力)、2026年AI capex 7,000億ドル テープ(本予測が名指すコスト圧力前提条件であり、本予測が最初に表れる形と予想する service-delivery-model / pricing-structure 変更ではない)。capex 対電力コスト圧縮は構築中;本予測が予測する pricing-structure シフトは本日のニュースで未だ可視化されていない。

**RL / LLM ベースのアプローチが現実世界現象の予測性能を向上させる** 予測は本日直接的支持が薄い。最も近い類似例は arxiv 2604.20801(Synthesizing Multi-Agent Harnesses for Vulnerability Discovery)で、マルチエージェントハーネスを合成して 10 個の Chrome ゼロデイ(CVE-2026-5280 + CVE-2026-6297 を含む)を発見——これは LLM 駆動の現実システム脆弱性予測であり、対象がオープンドメイン予測ではなくセキュリティであるとはいえ本予測の精神に属する。ICLR 2026 outstanding-papers 波(Q-RAG マルチステップ検索 RL 訓練埋め込み、MedAgentGym エージェント生物医学訓練、「LLMs Get Lost In Multi-Turn Conversation」)は LLM-agent-improves-via-RL パターンが 2026 年に主流化していることの研究側証拠だが、scheduled-task 風の予測ベンチマーク発表は本日着地せず。
